В файле Test.cpp реализовано 2 вида функции определяющей четное ли число.
Входные данные: число
Выходные данные: булевое значение является ли число четным.

Представлен последний вариант для тестов скорости (входное число формата unsigned long long int)
для сравнения скорости работы моей функции и функции представленной в задании.

При int однозначном числе профилировщик выдает значения представленные на картинке тест 1

При int многозначном числе профилировщик выдает значения пердставленные на картинке тест 2

При unsigned long long int многозначном числе профилировщик выдает значения представленные на картинке тест 3


Как видно при сравнении моя функция использующая бинарную операцию И работает быстре и стабильнее функции 
представленной как пример в тестовом задании №1
